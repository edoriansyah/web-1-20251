# Kriteria & Ketentuan Project Website – Pemrograman Web 1

## 1. Tujuan Project

Project ini bertujuan untuk mengukur kemampuan mahasiswa dalam menerapkan dasar-dasar pengembangan web meliputi struktur HTML, styling CSS, interaktivitas JavaScript, serta penggunaan CSS Framework (Bootstrap atau Tailwind) untuk membangun website responsif dan modern.

---

## 2. Bentuk Project

Mahasiswa diwajibkan membuat sebuah **website statis** dengan minimal **10 halaman** yang saling terhubung. Tema website bebas namun harus relevan dengan materi pemrograman web dasar (contoh: landing page, katalog produk, portofolio, blog sederhana, website informasi kampus, dsb).

---

## 3. Persyaratan Teknis

### A. Struktur & Konten

- Menggunakan minimal **10 halaman HTML** (contoh: Home, About, Contact, dll).
- Struktur HTML harus **semantik**: header, nav, main, section, article, footer.
- Setiap halaman wajib memiliki:
  - Header & navigasi
  - Konten utama
  - Footer

### B. Styling (CSS & Framework)

- Wajib menggunakan **CSS custom** dan **CSS Framework** sekaligus.
- Framework yang diperbolehkan:
  - Bootstrap, atau
  - Tailwind CSS
- Minimal 5 elemen menggunakan CSS custom.
- Minimal 5 komponen menggunakan framework.
- Website harus **responsif** (desktop & mobile).

### C. Interaktivitas JavaScript

Project harus memiliki minimal **5 fitur interaktif** menggunakan JavaScript murni. Contoh:

- Validasi form
- Toggle menu
- Modal popup
- Slider
- Dark mode toggle
- Event interaksi (click, hover, scroll)

### D. Media

Setiap halaman harus memuat minimal:

- 1 gambar
- 1 link internal
- 1 link eksternal
- 1 elemen multimedia (opsional)

### E. Struktur Folder

Contoh struktur folder:

```
/project-name
│── index.html
│── about.html
│── contact.html
│── /css
│    └── style.css
│── /js
│    └── script.js
│── /images
│    └── ...
```

---

## 4. Ketentuan Pengumpulan

- Project dikumpulkan ke dalam repository GitHub.
- Wajib menyertakan file **README.md** berisi:
  - Judul project
  - Daftar anggota kelompok
  - Deskripsi website
  - Daftar fitur
  - Teknologi yang digunakan
  - Screenshot tampilan

---

| Aspek                               | Bobot |
| ----------------------------------- | ----- |
| Struktur HTML & kelengkapan halaman | 20%   |
| Desain UI/UX & styling              | 25%   |
| Responsiveness                      | 15%   |
| Fitur JavaScript                    | 20%   |
| Kerapihan kode & struktur folder    | 10%   |
| Kreativitas & kesesuaian tema       | 10%   |

---

## 6. Ketentuan Tambahan

- Dilarang menggunakan template jadi dari internet.
- Tidak menggunakan backend atau database.
- Konten harus original dan tidak mengandung unsur negatif (SARA, kekerasan, pornografi).
- Dikerjakan secara kelompok (anggota kelompok sesuai arahan asisten dosen).
- Plagiasi akan mendapatkan nilai **0**.
