
# Workshop Praktikum JavaScript – Basics Part 2

## Bagian 1 — Conditionals

### 1.1 If–Else Sederhana
Buat program yang meminta input nilai ujian (0–100), lalu tampilkan kategori:
- A (≥ 90)
- B (≥ 80)
- C (≥ 70)
- D (≥ 60)
- E (< 60)

### 1.2 Switch Case
Buat program yang menerima angka 1–7 dan menampilkan nama hari:
1 → Senin, …, 7 → Minggu  
Selain itu: “Input tidak dikenal”

### 1.3 Ternary Operator
Gunakan ternary operator untuk menentukan apakah seseorang boleh membuat KTP (usia ≥ 17).

---

## Bagian 2 — Loops

### 2.1 For Loop
Tampilkan angka 1–20 menggunakan loop `for`.

### 2.2 While Loop
Tampilkan hitung mundur dari angka 10–1 menggunakan `while`.

### 2.3 Do–While Loop
Tampilkan “Halo!” minimal sekali meskipun kondisi awal salah.

### 2.4 Break & Continue
Buat loop 1–10 dengan syarat:
- Skip angka 5 (continue)
- Stop pada angka 8 (break)

---

## Bagian 3 — Functions

### 3.1 Function Declaration
Buat fungsi `luasPersegi(sisi)` yang mengembalikan luas persegi.

### 3.2 Function Expression
Buat function expression bernama `greet` untuk menyapa nama pengguna.

### 3.3 Arrow Function
Buat arrow function bernama `jumlah(a, b)`.

### 3.4 Default Parameter & Rest Parameter
1. `salam(nama = "Mahasiswa")`  
2. `jumlahSemua(...nums)` untuk menjumlahkan semua angka.

---

## Bagian 4 — Objects

### 4.1 Object Literal
Buat object `mahasiswa` berisi nama, nim, jurusan, dan method `tampilkanData()`.

### 4.2 new Object()
Buat object `mobil` menggunakan `new Object()` berisi brand, tahun, dan warna.

### 4.3 Modifikasi Object
- Ubah warna mobil  
- Tambahkan property `mesin`

---

## Bagian 5 — Arrays

### 5.1 Membuat Array
Buat array `buah = ["apel", "mangga", "pisang"]`.  
Tampilkan elemen pertama dan panjang array.

### 5.2 Menambah Elemen
Tambahkan elemen menggunakan:
- `push()`
- `buah[buah.length] = ...`

### 5.3 Looping Array
Tampilkan seluruh elemen array `buah` dengan loop.

### 5.4 Array toString
Tampilkan array dalam bentuk string.

### 5.5 Array Berisi Berbagai Tipe
Buat array dengan kombinasi string, angka, boolean, object, dan array.

Buat program daftar belanja dengan fitur:

2. Menghapus item  
3. Menampilkan seluruh daftar  

Gunakan minimal:
